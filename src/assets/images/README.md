# Profile Image Placeholder

This directory contains the profile image for the hero section.

## Required Image:
- **Filename**: `profile-photo.jpg`
- **Size**: Minimum 400x400px (square aspect ratio recommended)
- **Format**: JPG, PNG, or WebP
- **Quality**: High resolution for crisp display

## Image Guidelines:
- Professional headshot or portrait
- Good lighting and clear face visibility
- Square crop works best (will be displayed as circle)
- High contrast background recommended
- File size: Keep under 500KB for optimal loading
